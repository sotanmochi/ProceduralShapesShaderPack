## Tested Environment
- Unity 2020.3.18f1
- Universal Render Pipeline 10.6.0
- Shader Graph 10.6.0

## How to install
```
// manifest.json
{
  "dependencies": {
    "jp.sotanmochi.procedural-shapes-shader-pack": "https://github.com/sotanmochi/ProceduralShapesShaderPack.git?path=Packages/ProceduralShapesShaderPack",
    ...
  }
}
```
